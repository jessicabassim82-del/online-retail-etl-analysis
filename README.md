# Online Retail ETL Analysis

## Project Overview
This project demonstrates an ETL pipeline for processing an online retail dataset. The pipeline extracts raw data, cleans and transforms it using Python and Pandas, loads it into an SQLite database, and performs SQL queries to generate business insights.

## Tech Stack
- Python
- Pandas
- SQLite
- SQL
- Jupyter Notebook

## Dataset
Online Retail Dataset from Kaggle.

## ETL Steps
1. Extract data from CSV
2. Transform data (handle missing values, format dates)
3. Load data into SQLite database
4. Run SQL queries for business insights

## Insights
- Top selling products
- Monthly revenue trends
- Customer purchasing behavior
